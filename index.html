Meeko

A platform for implementing component-based systems and data analysis

  • GitHub Profile

Welcome to Meeko.

Meeko is a tiny single-node platform as a service that makes it very easy to build systems that consist of multiple components (in this case processes) that communicate with each other. The platform exports various services that the components (called agents) can use to exchange data. Each service implements one of the popular communication patters, e.g. publish-subscribe or request-reply, or some other functionality that is useful for the agents.

The platform consists of three basic component:

  • meekod is the core component (process) that contains the component supervisor and communication broker.
  • meeko is a command line management utility that can be used to install new components into the system, then configure and start them.
  • go-meeko is the Go client library that can be used by the components to connect to all the communication services. It makes the communication very simple and friendly.
